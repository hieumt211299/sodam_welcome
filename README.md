# Soldam Market Website

A fullscreen welcome page for Soldam Market.

## Setup Instructions

1. Save your Soldam Market image as `soldam-market.png` in this directory
2. The image should be the purple gradient background with "Welcome to SOLDAM MARKET" text
3. Open `index.html` in a web browser to view the site

## Files

- `index.html` - Main HTML file with fullscreen layout
- `styles.css` - CSS styling for fullscreen background and responsive design
- `soldam-market.png` - (You need to add this) The background image

## GitHub Pages Hosting

This site is ready to be hosted on GitHub Pages. Simply push to GitHub and enable Pages in repository settings.

## Features

- Fullscreen background image
- Responsive design for mobile devices
- Modern CSS with proper typography
- Ready for GitHub Pages deployment# sodam_welcome
